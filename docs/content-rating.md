## Notes for reviewers

- GameOn is a sports match organiser — users create pickup games and others join.
- No in-app purchases, no ads, no gambling.
- User-generated content is limited to: username, bio text, profile photo, match titles/descriptions.
- Location data: only match venue coordinates (entered by creator), not continuous GPS tracking.
- Minimum age: 13 (enforced in Terms of Service, not in-app).
- Account deletion available in Settings.
